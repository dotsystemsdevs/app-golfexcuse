# Golf Excuse Generator

Mobile app that generates random golf excuses. Pick optional weather (Windy, Rain, Cold, Hot), tap **Generate Excuse**, then copy and share.

![App logo](assets/logo.png)

## Features

- One-tap random excuse from 150+ options
- Optional weather prefix (Windy, Rain, Cold, Hot)
- Copy to clipboard with one tap
- Rate the app & legal links (Privacy, Terms) in the footer
- In-app update check (Expo Updates)
- Accessibility support (VoiceOver / TalkBack)

## Tech

- **React Native** (Expo SDK 54)
- **iOS & Android** (portrait, safe area)
- No backend; all data is local (e.g. “asked for review” flag)

## Run locally

```bash
npm install
npm run ios    # or: npm run android
```

Requires Node.js, npm, and Xcode (iOS) or Android Studio (Android).
